# ☕ Altura Café - Sitio Web Oficial

¡Bienvenido al repositorio de **Altura Café**! Este es el sitio web oficial para una cafetería de especialidad ubicada en La Paz, Bolivia. La plataforma está diseñada como una landing page moderna, minimalista y completamente responsiva, optimizada para ofrecer una excelente experiencia de usuario e incentivar los pedidos rápidos mediante canales digitales.

## 🎨 Características del Diseño

El proyecto cuenta con un diseño estético de alta fidelidad enfocado en el rubro gastronómico premium:
*   **Sección Hero de Alto Impacto:** Una presentación cinematográfica con imágenes integradas, tipografía elegante y un badge interactivo de disponibilidad en tiempo real (*"Abierto Ahora"*).
*   **Identidad de Marca Sólida:** Paleta de colores personalizada basada en tonos tierra, maderas y cafés configurada directamente mediante el archivo de estilos dinámicos.
*   **Menú Digital Destacado:** Tarjetas interactivas con efectos dinámicos de elevación (`hover:-translate-y-1`) que exponen los productos estrella de la cafetería y sus precios en moneda local (Bs).
*   **Sección de Ubicación Dinámica:** Mapa interactivo de Google Maps integrado junto a tarjetas informativas estilizadas para horarios y dirección exacta.
*   **Prueba Social (Testimonios):** Una sección dedicada a las opiniones de los clientes maquetada en una cuadrícula limpia.
*   **Llamado a la Acción (CTA) Eficaz:** Botones interactivos optimizados para redireccionar al usuario directamente a realizar pedidos a través de WhatsApp.

## 🛠️ Tecnologías y Librerías Utilizadas

El sitio web fue construido bajo un enfoque puramente Frontend moderno a través de cargas CDN estables, asegurando una velocidad de carga óptima sin necesidad de empaquetadores complejos:

*   **HTML5 Semántico:** Estructura limpia y accesible utilizando etiquetas semánticas (`<section>`, `<article>`, `<footer>`).
*   **Tailwind CSS (v3.x):** Utilizado para todo el maquetado responsivo, espaciados y paleta de colores personalizada mediante inyección en el objeto `tailwind.config`.
*   **AOS (Animate On Scroll v2.3.1):** Integrado para añadir dinamismo mediante efectos de aparición fluida de los elementos a medida que el usuario navega la página.
*   **Google Fonts:** Uso combinado de tipografías seleccionadas:
    *   `Playfair Display` (Serif): Utilizada exclusivamente para los títulos elegantes (`.title`).
    *   `Inter` (Sans-serif): Utilizada para el cuerpo de texto, garantizando máxima legibilidad.
